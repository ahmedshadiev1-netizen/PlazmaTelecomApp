# Как посмотреть логи сборки

## Способ 1: Скачать артефакт build.log (самый простой)

1. **Откройте страницу Actions:**
   - Перейдите на https://github.com/ahmedshadiev1-netizen/PlazmaTelecomApp/actions

2. **Выберите нужную сборку:**
   - Найдите нужную сборку (например, #22, #23 и т.д.)
   - Нажмите на неё

3. **Скачайте артефакт:**
   - Прокрутите страницу вниз до секции **"Artifacts"** (Артефакты)
   - Вы увидите артефакт **"build-logs"**
   - Нажмите на него, чтобы скачать ZIP-архив
   - Распакуйте архив - внутри будет файл `build.log`

4. **Откройте build.log:**
   - Откройте файл в любом текстовом редакторе
   - Ищите строки с "ERROR", "FAILED", "Exception"

## Способ 2: Посмотреть логи прямо в GitHub (быстро)

1. **Откройте страницу сборки:**
   - https://github.com/ahmedshadiev1-netizen/PlazmaTelecomApp/actions

2. **Выберите нужную сборку**

3. **Откройте шаг "Build APK":**
   - В списке шагов найдите "Build APK"
   - Нажмите на него, чтобы развернуть логи

4. **Ищите ошибки:**
   - Прокрутите вниз до секции "BUILD LOG (last 100 lines)"
   - Или найдите секцию "=== ERRORS ==="
   - Используйте поиск (Ctrl+F) и ищите: "error", "failed", "exception"

## Способ 3: Посмотреть полный лог шага Build APK

1. **Откройте сборку в GitHub Actions**

2. **Откройте шаг "Build APK"**

3. **Прокрутите вниз:**
   - В конце лога будет показан полный вывод команды `./gradlew assembleDebug`
   - Все ошибки будут видны там

## Что искать в build.log

Ищите строки с:
- `BUILD FAILED`
- `ERROR`
- `Exception`
- `FAILURE`
- `Task :app:compileDebugJavaWithJavac FAILED`
- `Task :app:mergeDebugResources FAILED`

## Пример расположения ошибок

Обычно ошибки находятся:
1. **В конце файла** - последние 50-100 строк
2. **После строки "BUILD FAILED"**
3. **В секции "Caused by:"** - там указывается причина ошибки

## Если build.log не найден

Если артефакт "build-logs" отсутствует, значит:
- Сборка не дошла до шага "Build APK"
- Или шаг "Build APK" упал до создания лога
- В этом случае смотрите логи шага "Build APK" прямо в GitHub (Способ 2)
