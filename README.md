# Плазмателеком - Мобильное приложение

Мобильное приложение для абонентов Плазмателеком с полным функционалом управления услугами.

## Возможности

- **Личный кабинет** - просмотр договоров и баланса
- **Управление услугами** - интернет, TVIP, видеонаблюдение
- **Обещанные платежи** - пополнение баланса 50-100 рублей
- **Push-уведомления** - важные события
- **Свайп-навигация** - удобное переключение между договорами и услугами

## Установка

### Требования
- Node.js 16+
- React Native CLI
- Android Studio (для Android)
- Xcode (для iOS)

### Установка зависимостей
```bash
npm install
```

### Запуск на Android
```bash
npx react-native run-android
```

### Запуск на iOS
```bash
npx react-native run-ios
```

## Структура приложения

### Главный экран
- **Имя абонента** вверху
- **Свайп по договорам** - горизонтальная прокрутка
- **Баланс и действия** - пополнение, история
- **Учетные записи** - услуги под выбранным договором

### Карточки услуг

#### Интернет
- Логин (MAC адрес)
- Статус подключения
- Скорость
- Адрес подключения
- Periodic Services (подпункт)

#### TVIP
- Описание пакета
- Логин и пароль
- Статус

#### Видеонаблюдение
- Детальное описание услуг
- Компоненты и цены
- Итоговая стоимость

## API интеграция

Приложение подключается к backend API:
- `POST /auth/login` - аутентификация
- `GET /contracts` - список договоров
- `GET /accounts` - учетные записи
- `POST /contracts/{id}/promise-payment` - обещанные платежи
- `GET /accounts/{id}/periodic-services` - периодические услуги

## Push-уведомления

Настроены уведомления для:
- Низкий баланс
- Новые платежи
- Блокировка услуг
- Технические работы

## Дизайн

- **Цвета:** Черный и зеленый (сдержанные тона)
- **Интуитивный интерфейс** для обычных пользователей
- **Адаптивный дизайн** для всех экранов
- **Свайп-навигация** для удобства

## Разработка

### Структура файлов
```
src/
├── components/     # Переиспользуемые компоненты
├── screens/       # Экраны приложения
├── services/      # API сервисы
```
